# 🪐 Ferox OS

> **The perfect balance of aesthetics and peak performance.** > A lightweight, privacy-focused operating system concept built on a solid LTS foundation, stripped of background telemetry, and designed to just work.

---

## ✨ Overview

**Ferox OS** is a modern Linux distribution concept designed for power users, gamers, and privacy enthusiasts who refuse to compromise on visual elegance. It features a stunning **"Liquid Glass" (Glassmorphism)** desktop environment, a custom interactive cursor, and an intelligent context menu system that adapts to your actions.

---

## 🚀 Key Features

* **🍃 Absolute Lightness:** No background telemetry, forced trackers, or heavy bloatware services. System resources are strictly reserved for what you run.
* **🎮 Native Gaming Out-of-the-Box:** Pre-configured with Steam, Proton compatibility layer, and the latest GPU drivers. Seamless gaming in a single click.
* **🔒 Built-In Privacy:** Automated DNS-over-HTTPS encryption, strict firewall rules, and absolute tracking prevention.
* **🎛️ Intelligent Context Menu:** A custom-engineered context menu that alters its state dynamically:
    * *Text Selected:* Shows only the **"Copy Text"** option for a distraction-free workflow.
    * *No Selection:* Shows OS controls like **"Specifications"**, **"Download ISO Image"**, and **"Reload Page"**.
* **🔔 Liquid Glass Toast Notifications:** Beautiful, top-down push notifications with smooth CSS transitions, glowing borders, and backdrop-blur effects.
* **🖱️ Adaptive Kinetic Cursor:** A dual-element custom cursor with a fluid, glassmorphic follower that expands and reacts when hovering over interactive elements.

| Parameter | Specification |

| **Package Base** | Ubuntu 26.04 LTS |
| **Linux Kernel** | Kernel 7.0 (Zen patch optimized) |
| **Desktop Environment** | GNOME 48 (Custom Shell & Blur-My-Shell integration) |
| **Architecture** | x86_64 / amd64 |
| **Init System** | systemd |

### 🖥️ Hardware Requirements

* **CPU:** Any 64-bit dual-core processor (2.0 GHz+)
* **RAM:** 4 GB minimum (8 GB recommended for gaming/multitasking)
* **Storage:** 25 GB free space (SSD strongly recommended)

## 💻 Tech Stack (Web Concept)

This interactive web-concept showcase is built with:
* **HTML5** — Semantic, clean structure.
* **CSS3** — Advanced animations, `@keyframes` drift, custom variables, `backdrop-filter` effects, and responsive design.
* **Vanilla JS** — Lightweight, zero-dependency scripting for custom cursor tracking, dynamic context-menu switching, and state management for modal windows and notifications.
