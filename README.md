# 🦊 Ferox OS

**Ferox OS** is a lightweight, stylish, and fast Linux distribution based on Ubuntu 26.04 LTS. It combines the power and stability of Ubuntu with a unique aesthetic and minimalist approach. Ferox OS is built for those who value speed, control, and beauty.

## ✨ Features

- 🚀 **Speed & Lightweight** — Optimized build with no bloatware, boots in seconds.
- 🎨 **Unique Style** — Custom dark theme **Orchis** and **Papirus** icon set for an aesthetic and cohesive look.
- 🛡️ **Reliability** — Based on stable and proven Ubuntu 26.04 LTS, ensuring compatibility with a vast ecosystem of software.
- 🖥️ **Gaming Ready** — Comes with necessary drivers and **Proton** support out of the box for Windows games.
- 🔒 **Security** — Minimal digital footprint and timely security updates.
- ⚡ **Modern Stack** — Linux kernel 7.0, GNOME 48, systemd.

## 🛠️ Installation

### 1. Download the ISO

Download the latest version of Ferox OS from the [Releases](https://github.com/your-username/ferox-os/releases) page.

### 2. Create a Bootable USB Drive

Use [Rufus](https://rufus.ie/) (Windows) or `dd` (Linux) to write the ISO to a USB drive.

**Example for Linux:**
```bash
sudo dd if=ferox-os-1.0-desktop-amd64.iso of=/dev/sdX bs=4M status=progress
